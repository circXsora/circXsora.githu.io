圆环计划2025——始动!
Created 2020-07-13| Updated 2021-03-25|ProjectMagia
Post View:

Project Magia 2025

圆环计划2025是我计划在2025年开始制作一款以魔法少女小圆 & 魔法纪录为世界观的同人游戏，灵感来源于twitter画师大大ほし的塞尔达风格的魔法纪录。

基本架构

这个游戏应该有类似与女神异闻录的Gal日常部分，与神浜、见泷源、风间野的各位魔法少女建立关系，以JRPG的主角团队为概念，主角团队和其他各个魔法少女都可以发展关系，触发各种事件，最终可以达成恋爱关系，也可以撮合魔法少女们达成恋爱关系（火焰纹章风花雪月的机制），大部分魔法少女都能邀请至主角团队并操作她来和某个魔法少女对话或发生事件，随着人物羁绊的加深，能力、技能、机制也会不断解锁从而强化人物。

战斗则采取魂like的3D动作、每个魔法少女都有不同的攻击方式、技能、半魔女化、魔女化（需要付出代价）、可以单人刷怪，也可以和队友一起触发组合技能。

当魔法少女陷入魔女化时，可以操纵其他魔法少女来净化她的灵魂晶石来解除魔女化，只要主角团队不全灭，则游戏不会失败。

最好能借鉴旺达与巨像的boss战演出。

整体画面风格应该接近塞尔达传说（ほし大大的设定）和女神异闻录。

游戏引擎为虚幻，开发语言为C#、C++。

规划

目前以这个为目标前进。附上大致计划：

2020初级阶段：学习使用Godot引擎来复刻几款2D小游戏（如细菌大战），熟悉GamePlay的制作，最终目标是制作一款丧尸生存类2D游戏（2D版腐烂国度，steam上的Zhelter）。学习C++、Cmake、Lua、Premake等基础的C++编译链接概念、更熟悉C++。

2020年阅读了几本基础技术书籍，并制定了接下来的学习计划，以及换了一份游戏行业开发的工作，C++暂时搁置吧，现在用不到，不用的话也会忘的。Godot也不会再进行了，对我来说远不如unity方便。通过职场的学习，意识到了自己很多方面的不足，接下来将要努力把不足的地方完善掉。

2021原型阶段：使用Unity复刻黑魂，深刻学习黑魂、血源的GamePlay，以及塞尔达传说的玩法。其中研究学习OpenGL、vulkan、dx12等图形API，学习游戏引擎架构代码以及一些开源游戏项目。

使用和学习Unity制作一款完整的2D游戏王同人RPG，学习Game Framework，了解框架的使用方法，并修改它。并学习Shader、AI、设计模式、catlikecoding等一些基础游戏制作理念，学习高质量的实时渲染。以及一些游戏设计入门书籍！完成一款游戏才是最重要的！

  • 游戏王同人2D RPG
  • Game Framework
  • Cat like coding

3月开启生产力计划，投放1.5w元资金用于购买提高生产力的产品如显卡，显示器，桌子，椅子，iPad等。

希望9月可以完成基本目标，以及找一份更好的工作。

2022深入阶段：学习使用虚幻引擎，深刻学习女神异闻录系统、黑魂3D战斗系统，以及把之前的项目搬到虚幻引擎上。学习最终幻想、旺达与巨像等更多gameplay。

2023数理阶段：打牢数学和物理知识，研究图形学、物理系统。

2024美术阶段：学习画画建模图形学，思考如何得到自己想要的图形效果。

2025启动阶段：开始制作魔法纪录的同人游戏。

这是一个大概的纲要，总之现在努力朝着这个目标前进，也许中间的道路很曲折，会花费比想象中更多的时间，但不放弃的话，总会有一些成果的，我还想原创一些魔法少女角色，联动邦邦、少女歌剧等。

想法是美好的，而幻想究竟能不能落入现实就得看自身了。

计划

计划分为多种计划：

  • 人生计划

  • 工作计划

  • 学习计划

  • 健身计划

  • 营销计划

工具

  • 十步学习法
  • 番茄工作法
  • 看板方法

首先是人生计划，圆环计划2025就是我提出的第一个五年计划（也许是第二个，我15岁的时候辍学 学习编程加入软件公司），这个是保证我拥有开发一款完整游戏的能力，并拥有相当程度的游戏设计能力与美术能力。而在22年的时候我想找机会去日本开发游戏。

然后就是第二个五年计划可能是努力实现财富自由，并去Square Enix开发最终幻想，当然，这看起来非常非常难，可能会成为第三个甚至第四个也说不定。

再来就是工作计划，工作中要成为专家，并且像企业一样思考，思考自己可以为他人提供什么样的产品和服务。我自己是想做一名Gameplay程序员，能出色完成游戏逻辑和充分理解游戏设计的意图，甚至自己设计游戏玩法并且实现。

但国内貌似并不太需要这样的游戏程序员。技美、主程、优化甚至后端更加吃香，所以在国内我可以先达到实现策划需求，然后对架构有一定了解，而且懂渲染知识，如果可以的话，希望能先主攻渲染这块，随着手机性能的提高，手游对画面的需求也会不断提高。而手游上的效果并不会多么复杂，所以也不需要太过高深的图形学知识。

学习计划则是首先把目前买的书看完，做好笔记，实现书中的例子效果。还有学习游戏框架，运用它，只有学以致用才能将学习效率发挥到最大。目标是完成一个完整的游戏，做出令人满意的画面效果（Shader）

健身计划则是购买哑铃，也可以先用两瓶矿泉水代替练习背肌坚持一个月没问题再买哑铃，然后是跑步机，但也可以先跑一个月再购买跑步机。最后目标是体重降到50kg以下。

营销计划，创建自己的对外技术博客，因为是中文的，所以最好在中文社区里，GitHub仅是自用记录。正式发布文章在gitbook上，国内暂时没想出什么好的网站，要么审美不达标，要么不方便，还有的要绑定手机号微信等，很烦，初步选定知乎、CSDN、博客园三个地方。并保持一周更新一篇技术文章，如果没想法，就挑个时间头脑风暴，想出20个主题。再来就是在b站、YouTube发布unity教学视频，将自己做的2D RPG同人游戏的教学发布出去，这个可以等自己的游戏有一定完成度再进行。

然后是个人品牌，自己的名字则用circXsora，也可以直接称sora，产品则一律以mga(magiasora)来命名。

另外工具的话则是最实用的看板方法和番茄工作法，工作日每天保证两个番茄时间，搬家后增加到4个，休息日保证10个番茄时间，尽量达到12个。看板方法则记录周计划和日计划，月计划和季度计划则记录到本子上。（使用Kanbanflow）

3月-4月因为该死的加班番茄和看板停了一段时间。。。

完整的十步学习法来学习新领域的东西，如摄影等，简化版十步学习法则学习已经有基础的东西。否则会浪费比较多时间在找资源里面。
